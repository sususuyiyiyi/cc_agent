## 🚀 启用真实搜索

### 步骤 1: 配置 WebSearch API

注册并获取 Brave Search API Key：
1. 访问 https://brave.com/search/api/
2. 注册账号
3. 获取免费 API Key（每月2000次）

配置 API：
```bash
python3 configure.py --websearch YOUR_API_KEY
```

### 步骤 2: 更新配置

配置会自动保存到 `config/config.yaml`：

```yaml
websearch:
  enabled: true
  search_engine: default
  max_results: 10
  timeout: 30
```

### 步骤 3: 测试

```bash
# 测试新闻获取
python3 -c "from news_agent import NewsAgent; agent = NewsAgent(); agent.run()"
```
