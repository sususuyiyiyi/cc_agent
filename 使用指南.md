# CC Agent 使用指南

欢迎使用 CC Agent 个人生活助理团队！

## 🚀 快速开始

### 交互式模式

```bash
python3 run_agents.py
```

### 命令行模式

```bash
# 运行资讯su
python3 run_agents.py --news

# 运行营养师su
python3 run_agents.py --wellness

# 运行复盘su
python3 run_agents.py --review

# 运行所有agents
python3 run_agents.py --all

# 运行测试
python3 run_agents.py --test

# 查看状态
python3 run_agents.py --status
```

## 📦 三个 Agents 介绍

### 1. 资讯su (news-su)

**触发时间**: 每天早上 8:00

**功能**:
- 问候用户（早上好）
- 获取最新的 AI 资讯和新闻
- 总结成新闻简报
- 创建日期文件夹保存新闻
- 发送到飞书（可选）

**输出文件**: `data/news/年/月/日/今日新闻.md`

---

### 2. 营养师su (wellness-su)

**触发时间**: 每天早上 8:30

**功能**:
- 获取当天天气信息
- 根据天气给出饮食建议
- 根据天气给出穿搭建议
- 保存建议到历史记录

**输出文件**: `data/wellness/年/月/日/建议.md`

---

### 3. 复盘su (review-su)

**触发时间**: 每天晚上 20:00

**功能**:
- 主动询问用户今天做了什么
- 根据用户回答生成日报
- 保存日报到文件夹
- 发送到飞书（可选）

**输出文件**: `data/reviews/年/月/日/日报.md`

## ❓ 常见问题

### Q: 如何修改 agents 的触发时间？

A: 在配置文件中修改定时任务设置，或者在代码中修改调度逻辑。

### Q: 数据存储在哪里？

A: 所有数据存储在 `data/` 目录下，按日期组织。

### Q: 如何导出数据？

A: 数据以 Markdown 格式存储，可以直接复制或使用工具批量导出。

### Q: 如何添加新的 agent？

A: 参考 `skills/` 目录下的示例，创建新的 SKILL.md 文件和相关目录结构。
