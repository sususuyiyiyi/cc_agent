## 🔧 修改配置

### 修改执行时间
编辑 `config/config.yaml`，修改对应 agent 的 `time` 字段。

示例：
```yaml
news_su:
  enabled: true
  time: "09:00"  # 改为 9:00
```

### 修改时区
编辑 `config/config.yaml`，修改 `scheduling.timezone` 字段。

示例：
```yaml
scheduling:
  timezone: Asia/Shanghai  # 北京时间
  # 或使用其他时区:
  # timezone: UTC  # 协调世界时
  # timezone: America/New_York  # 纽约时间
```

### 启用/禁用特定 Agent
编辑 `config/config.yaml`，修改对应 agent 的 `enabled` 字段。

示例：
```yaml
news_su:
  enabled: false  # 禁用资讯su
```
