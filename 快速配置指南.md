# CC Agent 快速配置指南

## 🎯 5 分钟快速配置

### 步骤 1: 安装依赖（1分钟）

```bash
cd /Users/sususu/cc_agent

# 运行安装脚本
./install.sh
```

### 步骤 2: 配置 WebSearch（2分钟）

1. 访问 https://brave.com/search/api/
2. 注册账号并获取 API Key
3. 运行配置命令：

```bash
python3 configure.py --websearch YOUR_API_KEY
```

### 步骤 3: 配置天气 API（2分钟）

**推荐使用 OpenWeatherMap（免费额度较大）：**

1. 访问 https://openweathermap.org/api
2. 注册账号并获取免费 API Key
3. 运行配置命令：

```bash
python3 configure.py --weather openweathermap YOUR_API_KEY
```

### 步骤 4: （可选）配置飞书（5分钟）

1. 在飞书群聊中：
   - 点击右上角 "..."
   - 选择 "设置" > "群机器人"
   - 点击 "添加机器人"
   - 创建机器人并复制 Webhook URL

2. 运行配置命令：

```bash
python3 configure.py --feishu YOUR_WEBHOOK_URL
```

### 步骤 5: （可选）启用定时任务

```bash
python3 configure.py --enable-scheduling
```

## ❓ 常见问题

### Q: 必须配置所有项吗？

A: 不是。
- **必需**: WebSearch（用于新闻）
- **推荐**: 天气 API（用于健康建议）
- **可选**: 飞书集成（用于消息推送）
- **可选**: 定时任务（用于自动运行）

### Q: 如何获取 API Key？

A:
- **Brave Search**: https://brave.com/search/api/（免费，每月2000次）
- **OpenWeatherMap**: https://openweathermap.org/api（免费，每月100万次）
- **和风天气**: https://dev.qweather.com/（国内服务）

### Q: 配置后如何测试？

A:
```bash
# 测试各个功能
python3 run_agents.py --news      # 测试新闻
python3 run_agents.py --wellness  # 测试健康建议
python3 run_agents.py --review    # 测试复盘

# 测试飞书
python3 feishu_client.py --test
```

### Q: 如何修改配置？

A: 直接编辑 `config/config.yaml` 文件，或使用配置工具：

```bash
python3 configure.py
```

## 📝 配置文件说明

### config/config.yaml

主配置文件，包含：
- 用户设置（姓名、时区、位置）
- Agent 调度配置
- WebSearch 配置
- 天气 API 配置
- 飞书集成配置
- 日志配置
- 用户偏好

### config/mcp_config.json

MCP 工具配置文件，包含：
- filesystem（文件系统访问）
- websearch（Web 搜索）
- fetch（网页抓取）
- postgres（数据库，可选）

## 🔧 高级配置

### 修改定时任务时间

编辑 `config/config.yaml`:

```yaml
scheduling:
  enabled: true
  news_su:
    enabled: true
    time: "08:00"  # 修改这里
  wellness_su:
    enabled: true
    time: "08:30"  # 修改这里
  review_su:
    enabled: true
    time: "20:00"  # 修改这里
```

### 添加新闻源

编辑 `skills/news-su/SKILL.md`，在参考部分添加你的新闻源。

### 自定义天气位置

编辑 `config/config.yaml`:

```yaml
user:
  location:
    city: Shanghai
    country: China
    lat: 31.2304
    lon: 121.4737
```
