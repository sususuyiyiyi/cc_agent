
### 添加新闻源
```bash

python3 scripts/manage_news_sources.py --add https://example.com

# 优先源
python3 scripts/manage_news_sources.py --add https://example.com --priority
```

### 删除新闻源
```bash
python3 scripts/manage_news_sources.py --remove https://example.com
```

### 设置/取消优先源
```bash

# 取消优先源
python3 scripts/manage_news_sources.py --priority https://example.com --unset-priority
```
