## ⏳ 待完成的配置

### 必需配置

#### 1. WebSearch API Key（用于新闻获取）

**目的**: 获取最新的 AI 资讯和新闻

**配置步骤**:
1. 访问 https://brave.com/search/api/
2. 注册账号并获取 API Key
3. 运行配置命令：

```bash
python3 configure.py --websearch YOUR_API_KEY
```

### 推荐配置

#### 2. 天气 API Key（用于健康建议）

**目的**: 获取实时天气信息

**选项 A - OpenWeatherMap（推荐）**
```bash
python3 configure.py --weather openweathermap YOUR_API_KEY
```

**选项 B - 和风天气**
```bash
python3 configure.py --weather qweather YOUR_API_KEY
```

### 可选配置

#### 3. 飞书集成（用于消息推送）

**目的**: 将新闻、健康建议、日报发送到飞书群聊

**配置步骤**:
1. 在飞书群聊中添加群机器人
2. 复制 Webhook URL
3. 运行配置命令：

```bash
python3 configure.py --feishu YOUR_WEBHOOK_URL
```

#### 4. 定时任务（用于自动运行）

**目的**: 每天 8:00, 8:30, 20:00 自动运行各个 agents

```bash
python3 configure.py --enable-scheduling
```

## 🚀 如何开始使用

### 快速开始

1. **查看系统状态**
```bash
python3 run_agents.py --status
```

2. **手动运行 agents**
```bash
python3 run_agents.py --news      # 资讯su
python3 run_agents.py --wellness  # 营养师su
python3 run_agents.py --review    # 复盘su
```

3. **交互式运行**
```bash
python3 run_agents.py
```

### 启动调度器

配置定时任务后，启动调度器：
```bash
python3 scheduler.py
```
