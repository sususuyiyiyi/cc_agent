## 🚀 如何使用

### 手动获取新闻
```bash
# 从优先源获取
python3 scripts/fetch_news.py --priority

# 从所有源获取
python3 scripts/fetch_news.py --all

# 限制新闻数量
python3 scripts/fetch_news.py --limit 5
```

### 自动获取
配置定时任务后，每天 08:00 自动获取新闻并发送到飞书。
