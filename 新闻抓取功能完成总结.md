## 🔧 启用真实搜索（可选）

### 配置 WebSearch API
1. 注册 Brave Search API: https://brave.com/search/api/
2. 获取免费 API Key（每月2000次）
3. 配置 API:

```bash
python3 configure.py --websearch YOUR_API_KEY
```

配置后系统会自动使用真实搜索获取新闻。
