## 🚀 如何使用

### 手动发送消息

```bash
# 发送新闻简报
python3 run_agents.py --news

# 发送健康建议
python3 run_agents.py --wellness

# 发送日报
python3 run_agents.py --review

# 发送所有消息
python3 run_agents.py --all
```

### 自动定时发送

配置定时任务后，agents 会自动在指定时间发送消息：

- **08:00** - 新闻简报
- **08:30** - 健康建议
- **20:00** - 日报

启用定时任务：

```bash
python3 configure.py --enable-scheduling
```
